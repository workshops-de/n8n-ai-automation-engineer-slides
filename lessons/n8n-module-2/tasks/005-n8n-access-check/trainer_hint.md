# Trainer Notes

## Pre-Workshop (3-5 days before)

- Send invitation emails to all registered participants
- Include login credentials for https://workshops-de-training.app.n8n.cloud/
- Prepare a backup list of participants who haven't confirmed access

## On Workshop Day 1

- Ask who still needs access activation
- Have credentials ready for on-the-spot activation
- Budget 5-10 minutes at the start for access issues
- Test the instance is running and accessible

## Common Issues

- **Spam folder**: Many corporate email systems filter automated invites
- **VPN/Firewall**: Some company networks may block cloud instances
- **Wrong credentials**: Have a reset process ready

## Activation Process

If someone needs immediate activation:
1. Create user account in N8N cloud admin
2. Send temporary password
3. Have them log in and change password
4. Usually takes 2-3 minutes
