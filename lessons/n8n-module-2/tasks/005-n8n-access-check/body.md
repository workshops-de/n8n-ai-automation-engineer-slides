# Preparation: N8N Access Check

## Task

Verify that you have access to our N8N learning environment. We send it 3-5 days before the workshop starts.

### Step-by-Step Guide

1. **Check Your Email**
   - Look for an invitation email from workshops.de
   - Subject might include "You have been invited to n8n" or similar
   - Check your spam folder if you don't see it in your inbox

2. **Access the N8N Instance**
   - Open your browser and navigate to: **https://workshops-de-training.app.n8n.cloud/**
   - You should be able to log in with your provided credentials

3. **Verify Login**
   - Try logging in to the N8N instance
   - If successful, you should see the N8N dashboard
   - Familiarize yourself with the interface (optional)

## What if I Didn't Receive an Invitation?

**Don't worry!** If you haven't received an invitation:

- **Before the workshop**: Contact us at the provided email address
- **On the first day**: We can activate your access immediately during the workshop
- It's quick and easy to set up, so you won't miss anything!

## Success Criteria

- [ ] I received the invitation email
- [ ] I can access https://workshops-de-training.app.n8n.cloud/
- [ ] I can log in successfully
- [ ] I see the N8N dashboard

## Need Help?

If you encounter any issues:
- Contact the workshop organizers
- Bring it up on the first day of the workshop
- We're here to help!
