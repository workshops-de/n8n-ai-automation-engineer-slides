# Preparation: Install DBeaver

**DBeaver Community** is a free database client. Install it before the module so you can test PostgreSQL connections and run SQL comfortably.

## Download

- Open the official download page: **[https://dbeaver.io/download/](https://dbeaver.io/download/)**
- Choose **DBeaver Community** and the package for your operating system.

## Installation (short guide)

- **macOS**: Download the DMG, open it, and drag **DBeaver** into **Applications**.
- **Windows**: Run the installer and follow the wizard.
- **Linux**: e.g. install the `.deb` with `sudo dpkg -i dbeaver-<version>.deb` or use the matching RPM — see the [download page](https://dbeaver.io/download/) for details.

## Success criteria

- [ ] Installer or archive downloaded from [dbeaver.io/download](https://dbeaver.io/download/)
- [ ] DBeaver installed and launched at least once successfully
- [ ] You see the DBeaver UI (a new/empty workspace is fine)
