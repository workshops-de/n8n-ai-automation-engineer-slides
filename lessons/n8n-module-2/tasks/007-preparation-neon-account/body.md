# Preparation: Neon account (neon.tech)

For the upcoming PostgreSQL exercises you need a free **Neon** account — Neon is a serverless Postgres provider. Create it before the module so we can start without delay.

## What is Neon?

[Neon](https://neon.tech) runs managed PostgreSQL databases in the cloud. You get a connection URL and can connect from n8n, DBeaver, and other tools.

## Step-by-step

### 1. Create an account

- Open [https://neon.tech](https://neon.tech) and choose **Sign up** (e.g. with GitHub, Google, or email).
- Confirm your email if verification is required.

### 2. Create a project (if you are starting fresh)

- After sign-in you can create a **project** and your first **PostgreSQL database** — default settings are fine for this course.

### 3. Keep your access handy

- Note which sign-in method you use for Neon.
- We will configure the connection string and details together in the session — what matters now is that the account exists and you can log in.

## Success criteria

- [ ] Account created on [neon.tech](https://neon.tech)
- [ ] Email confirmed (if required)
- [ ] You can sign in to the Neon dashboard
