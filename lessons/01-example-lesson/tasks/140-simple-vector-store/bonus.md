# Bonus Task

**Compare Retrieval Quality**: Experiment with different "Limit" values (1, 3, 5, 10) and see how it affects the agent's answers.

**Multiple Knowledge Bases**: Create multiple memory keys for different topics (e.g., `hr-policies`, `tech-docs`) and build an agent that can access all of them.

**Document Metadata**: Add metadata to your documents (e.g., category, date, author) and observe how it appears in retrieval results.

**Test Edge Cases**: Ask questions that aren't in the knowledge base and verify the agent responds appropriately.
